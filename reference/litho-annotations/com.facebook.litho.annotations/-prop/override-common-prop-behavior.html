overrideCommonPropBehavior

abstract fun overrideCommonPropBehavior(): Boolean

This may only be set to true if isCommonProp is also set to true. If true, then the behavior of it's common props will be overridden. If false, then the common prop will applied by the framework level as normal as well as any behavior that the component declares within the spec.

Return

true if the framework should not apply the common prop.
