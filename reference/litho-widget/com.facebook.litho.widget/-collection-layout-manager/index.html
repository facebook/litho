CollectionLayoutManager

@ExperimentalLithoApi
class CollectionLayoutManager(val layoutInfo: LayoutInfo, wrapInMainAxis: Boolean = false, wrapInCrossAxis: CrossAxisWrapMode = CrossAxisWrapMode.NoWrap)

A layout manager responsible for measuring and laying out collection components.

This class handles the measurement of CollectionRecyclerComponent and its children, supporting different wrapping behaviors in both main and cross axes.

Parameters

layoutInfo

Information about the layout configuration

wrapInMainAxis

Whether the collection should wrap its content in the main scrolling axis

wrapInCrossAxis

The wrapping behavior for the cross axis (perpendicular to scrolling)
