ComparableDrawable

interface ComparableDrawable : Equivalence<T>

Comparable Drawable allows drawables to be compared by explicitly implementing the isEquivalentTo. This allows drawables to be compared in a more meaningful way, instead of using equals which only checks if the references are equal.
