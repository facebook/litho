ComparableDrawableWrapper

abstract class ComparableDrawableWrapper : Drawable, ComparableDrawable, Drawable.Callback

Comparable Drawable Wrapper delegates all calls to its wrapped Drawable.

The wrapped Drawable must be fully released from any View before wrapping, otherwise internal Drawable.Callback may be dropped.
