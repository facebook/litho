DebugComponent

class DebugComponent

A DebugComponent represents a node in Litho's component hierarchy. DebugComponent removes the need to worry about implementation details of whether a node is represented by a or a ComponentLayout. The purpose of this class is for tools such as Stetho's UI inspector to be able to easily visualize a component hierarchy without worrying about implementation details of Litho.
