UnsafeModificationPolicy

enum UnsafeModificationPolicy : Enum<ComponentHost.UnsafeModificationPolicy>

This determines what is the actions to take if we detected an invalid modification of a ComponentHost.

This can happen for example if a client sets a click listener outside of a Litho specific codepath.

This method demonstrates how to use a Column with a specific style.

Example Kotlin code:

Column(style = Style.onVisible { event ->
  val view = event.content as? ComponentHost
  if(view != null) view.setOnClickListener { ... } // This an invalid usage
}
