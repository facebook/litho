StyleCompat

object StyleCompat

Backwards compatibility to allow Java code to define Styles used by Kotlin components

Usage:

new MyKotlinComponent(
    prop1,
    prop2,
    StyleCompat.flexShrink(0f)
        .marginDip(YogaEdge.ALL, 4f)
        .build())
