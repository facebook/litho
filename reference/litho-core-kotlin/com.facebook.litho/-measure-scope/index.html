MeasureScope

class MeasureScope(val layoutContext: LayoutContext<*>, val previousLayoutData: Any?)

The scope for the SimpleMountable measure method. Provides access to androidContext, previousLayoutData and utility methods that help return the appropriate MeasureResult.
